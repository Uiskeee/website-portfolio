CakeCakeCake!

2D Party Game for PlayStation 5

Overview

CakeCakeCake! is a competitive 2D party game built in C++ using a custom framework for the Playstation 5. It was made for my final year projects over a span of 3 months, in a team of 5, including 2 artists.

My Role

  • Project Manager
  • Lead Gameplay Programmer

Technical Challenges

The biggest challenge was shaping the currently 3D framework given to us to fit our 2D needs.

Tools Used

C++, Custom Engine, PlayStation SDK
