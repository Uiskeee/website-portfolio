CakeCakeCake!

2D Party Game for PlayStation 5

Overview

CakeCakeCake! is a competitive 2D party game built in C++ using a custom framework. Designed for PlayStation 5, the game focuses on chaotic multiplayer interactions and fast-paced mini-games.

My Role

  • Implemented core gameplay systems
  • Developed input handling for PS5 controllers
  • Optimized rendering performance

Technical Challenges

The biggest challenge was designing a modular event system that could handle rapid gameplay state transitions without frame drops.

Tools Used

C++, Custom Engine, PlayStation SDK
